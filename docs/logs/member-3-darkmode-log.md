## [التاريخ: 2026-07-08]

### ما تم إنجازه
- تدقيق index.css: تبيّن أنه يحتوي مسبقاً على متغيرات CSS + `@media (prefers-color-scheme: dark)` (وضع مظلم تلقائي حسب النظام)
- **قرار تم اتخاذه (بالتنسيق):** الإبقاء على `prefers-color-scheme` التلقائي بدون زر تبديل يدوي — لا حاجة لـ `ThemeContext.jsx` أو `localStorage` في هذه المرحلة
- **قرار تم اتخاذه (بالتنسيق):** اعتماد Tailwind classes مبنية على متغيرات CSS عبر `@theme` بدل استبدال الألوان مباشرة بـ `var(--x)`
- أضفت توكنز جديدة بـ index.css: `--bg-subtle`, `--text-muted`, `--warning-bg`, `--warning-border`, `--warning-text` (فاتح ومظلم) + ربطها بـ Tailwind عبر `@theme` (`bg-bg`, `text-text`, `border-border`, `text-accent`...الخ)
- حوّلت الألوان الثابتة إلى توكنز الثيم في:
  - `src/pages/Subject.jsx` (كانت كلها inline styles)
  - `src/components/Sidebar.jsx`
  - `src/pages/SubjectList.jsx`
  - `src/pages/StudyPlan.jsx`
  - `src/App.jsx` (خلفية أساسية فقط)

### مشكلات واجهتها
- لون الـ accent المستخدم بالمكونات كان `#2563eb` (أزرق) بينما `index.css` يعرّف `--accent: #aa3bff` (بنفسجي) كهوية الموقع الرسمية. تم توحيدها على `--accent` الموجود مسبقاً لضمان الاتساق البصري — يُرجى إبلاغ بقية الأعضاء إذا كان الأزرق مقصوداً كلون مختلف

### قرارات اتخذتها (ولماذا)
- الاكتفاء بـ prefers-color-scheme بدل بناء ThemeContext + localStorage: أبسط، ويغطي الاحتياج الحالي بدون تعقيد إضافي. يمكن ترقيته لاحقاً لتبديل يدوي إذا احتاج الفريق ذلك (سيتطلب حينها العودة لتعديل index.css وإضافة ThemeContext.jsx)
- استخدام Tailwind @theme بدل var() مباشرة: يقلل التكرار ويجعل الكود أنظف عبر كل الصفحات

### هل احتجت تعديل ملف مشترك؟ (نعم/لا) — إذا نعم، هل تم التنسيق مع الإدارة أولاً؟
- نعم. تم تعديل: App.jsx, Sidebar.jsx, SubjectList.jsx, Subject.jsx, StudyPlan.jsx
- تم التنسيق مسبقاً حول آلية التفعيل وأسلوب التحويل قبل البدء بالتنفيذ

### ملاحظات للأعضاء الآخرين
- ⚠️ **مهم للعضو 1 والعضو 2:** أي عنصر UI جديد تضيفونه (لوحة تحكم، أزرار تبديل دكتور...) الرجاء استخدام توكنز الثيم الجديدة بدل ألوان [#hex] ثابتة حتى لا يُكسر التوافق مع الوضع المظلم:
  - خلفية: `bg-bg` / `bg-bg-subtle`
  - نص: `text-text` / `text-text-h` (عناوين) / `text-text-muted` (نص خافت)

  - تمييز/روابط: `text-accent` / `bg-accent`
  - تنبيه: `bg-warning-bg` / `border-warning-border` / `text-warning-text`
- ⚠️ لم يتم بناء `ThemeContext.jsx` لأن الحاجة انتفت (قرار الفريق: تلقائي فقط). إذا تغيّر القرار مستقبلاً يجب تحديث هذا الملف والتنسيق مجدداً

- 🔴 **يجب إبلاغ الإدارة والعضو 4:** هذا النوع من التعارض (تعديلان متوازيان على نفس الملف المشترك) هو بالضبط ما حذّرت منه خطة الفريق. لو تكرر بملف آخر أو تلامس التعديلان فعلياً بنفس الأسطر، يجب أن يمر الدمج عبر الإدارة/العضو 4 وليس بشكل مباشر بين عضوين
